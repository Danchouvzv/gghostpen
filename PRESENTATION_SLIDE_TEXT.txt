РЕШЕНИЕ
────────
Style-Oriented Generation Pipeline

ML-пайплайн контролируемой генерации:
1. Style Profiler → извлекает метрики стиля (длина, тон, фразы)
2. Prompt Builder → превращает стиль в constraints для LLM
3. LLM Generator → генерирует с учётом ограничений
4. Post-Processor → корректирует под метрики автора
5. Style Scorer → оценивает качество (similarity 0.6–0.8)
